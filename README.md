# Flight System which Uses MYSQL Database

In this project we made a flight system. Modules that we have listed below:

1. Login
2. Search Flight
3. Select Flight
4. Create Ticket 
5. Change Ticket
6. Online Check In
 

In database we have 8 tables. Table names listed below:

1. bilet
2. binis_karti
3. fiyat
4. havayolu
5. koltuk
6. kullanici
7. sinif
8. student
